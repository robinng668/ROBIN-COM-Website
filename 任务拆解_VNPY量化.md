## 三、技术架构

### 3.1 系统架构

```
┌─────────────────────────────────────────┐
│              策略层                      │
│  (趋势策略 | 均值回复 | 轮动策略)        │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│              信号层                      │
│  (买入信号 | 卖出信号 | 仓位计算)        │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│              风控层                      │
│  (止损 | 止盈 | 仓位控制)                │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│              交易层                      │
│  (Futu API | 模拟/实盘)                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│              数据层                      │
│  (实时行情 | 历史数据 | 持仓数据)         │
└─────────────────────────────────────────┘
```

### 3.2 文件结构

```
quantitative-research/
├── data/
│   ├── realtime/      # 实时行情
│   ├── history/       # 历史数据
│   └── factors/      # 因子数据
├── strategies/
│   ├── trend/         # 趋势策略
│   ├── mean_rev/      # 均值回复
│   └── rotation/      # 轮动策略
├── backtest/
│   ├── engine.py      # 回测引擎
│   ├── optimizer.py   # 参数优化
│   └── analyzer.py    # 性能分析
├── trading/
│   ├── simulator.py   # 模拟交易
│   ├── executor.py    # 实盘执行
│   └── orders.py      # 订单管理
├── risk/
│   ├── position.py    # 仓位风控
│   ├── stoploss.py    # 止损风控
│   └── alert.py       # 预警系统
└── main.py            # 主入口
```
