## 二、1.1 市场整体概览 规范

### 2.1 数据采集要求

| 数据类型 | 来源 | 要求 |
|----------|------|------|
| 指数数据 | 实时行情API | 沪指/深成指/创业板/科创50 |
| 成交额 | 实时行情API | 每日成交量 |
| 资金流向 | 东方财富/同花顺 | 北向资金/主力资金 |
| 商品期货 | Yahoo Finance | 原油/黄金/铜 |
| 宏观数据 | Wind/ Bloomberg | GDP/CPI/利率 |

### 2.2 归因分析方法

**必须包含：**
1. **头部机构观点**：收集3-5家头部券商最新观点
2. **市场热点新闻**：当日最热10条财经新闻
3. **时事政治热点**：影响市场的重大政策/事件
4. **归因分析**：用一句话总结核心驱动因素

### 2.3 表格规范

| 列名 | 说明 | 示例 |
|------|------|------|
| 指标 | 数据名称 | 沪指收盘价 |
| 数值 | 最新数据 | 3,420点 |
| 涨跌幅 | 变化幅度 | +0.2% |
| 点评 | 一句话分析 | 科技股带动，指数分化 |

## 三、1.2 主要驱动因素 规范

### 3.1 内容要求

| 项目 | 要求 |
|------|------|
| 主题名称 | 明确的主题名 |
| 核心描述 | 200字以上详细描述 |
| 时间节点 | **必须精确到具体日期**，如"3月15日" |
| 数据支撑 | 必须有具体数据，不能用"较多"等模糊词 |
| 标的推荐 | 每个主题至少推荐2只标的 |
| 推荐理由 | 100字以上详细理由 |

### 3.2 禁止出现

❌ "两会召开在即" → ✅ "3月15日全国人大会议召开"  
❌ "近期" → ✅ "2026年3月"  
❌ "订单较多" → ✅ "订单同比增长50%"  
❌ "可能" → ✅ "预计"（需有依据）

### 3.3 表格规范

| 列名 | 说明 |
|------|------|
| 主题 | 驱动主题名称 |
| 描述 | 详细描述（含时间/数据） |
| 推荐标的 | 股票代码+名称 |
| 推荐理由 | 详细推荐逻辑 |

## 五、二、各行业分析 规范

### 5.1 行业选择

**至少覆盖10个重点行业，按景气度排序：**

| 景气度 | 行业 |
|--------|------|
| 第1 | AI算力/光通信 |
| 第2 | 半导体设备 |
| 第3 | 液冷温控 |
| 第4 | 游戏传媒 |
| 第5 | 创新药 |
| ... | ... |

### 5.2 行业核心逻辑

**300字以上，包含：**
1. 行业当前状态
2. 核心驱动因素
3. 未来发展趋势

### 5.3 五维度分析

**必须包含：**

| 维度 | 分析内容 |
|------|----------|
| 政策面 | 政策支持/限制具体内容 |
| 资金面 | 机构持仓/资金流向 |
| 基本面 | 业绩/订单/产能 |
| 催化剂 | 即将发生的利好事件 |
| 风险点 | 潜在风险 |

**每维度用星级表示景气度：⭐⭐⭐⭐⭐**

### 5.4 个股详表（核心标的）

**必须包含列：**

| 列名 | 说明 | 示例 |
|------|------|------|
| 标的 | 代码+名称 | 寒武纪 688256.SH |
| 最新价 | **实时价格** | 145.60元 |
| 评级 | 买入/增持/中性 | 买入 |
| 目标价 | 机构目标价 | 180元 |
| 万德2026E | Wind一致预期PE | 亏损 |
| 逻辑 | 推荐理由 | 国产AI芯片龙头 |
| 关联事项 | 重大事件 | 3/15华为发布会 |

### 5.5 行业小结

**100字以上，包含：**
1. 行业整体判断
2. 当前投资建议
3. 重点关注标的

## 六、个股数据要求

### 6.1 必须包含的数据

| 数据项 | 来源 | 更新频率 |
|--------|------|----------|
| 最新价 | Futu API | **实时** |
| 涨跌幅 | 计算得出 | 实时 |
| 万德2026E PE | Wind/估算 | 每日 |
| 目标价 | 卖方研报 | 每周 |
| 评级 | 卖方研报 | 每周 |

### 6.2 数据缺失处理

- **万德PE缺失**：标注"待出"或用可比公司估算
- **目标价缺失**：用PE估值法估算
- **最新价获取失败**：用昨收盘价并在备注标注
