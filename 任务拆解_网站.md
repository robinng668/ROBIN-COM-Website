## 三、技术方案

### 3.1 技术栈

| 用途 | 技术 |
|------|------|
| 前端框架 | Vue3 / React |
| 图表 | ECharts / TradingView |
| 行情数据 | Futu API / Baostock |
| 部署 | Cloudflare Pages |

### 3.2 数据流

```
Futu OpenD → API接口 → 前端展示 → Cloudflare Pages
                                    ↓
                              每日自动更新
```

---

## 四、产出标准

### 4.1 视觉标准

- 配色：深蓝色专业主题 (#0a1628)
- 字体：思源黑体 / PingFang SC
- 图表：专业金融风格
- 响应式：支持PC/平板/手机

### 4.2 功能标准

- 行情延迟 < 1秒
- 页面加载 < 2秒
- PDF在线预览
- 移动端适配
